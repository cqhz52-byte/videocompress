# 手机视频压缩工具

一个简单的 PWA 网页工具，用来压缩手机录制的大视频。

## 使用方式

1. 在当前目录启动静态服务器：

   ```powershell
   python -m http.server 5173
   ```

2. 打开浏览器访问：

   ```text
   http://localhost:5173
   ```

3. 选择或拖入视频，选择压缩档位，点击“开始压缩”，完成后下载压缩后的视频。

## 手机上安装

部署到 HTTPS 网站后，用手机浏览器打开网站：

- Android Chrome / Edge：菜单里选择“添加到主屏幕”或“安装应用”。
- iPhone Safari：分享按钮里选择“添加到主屏幕”。

## 说明

- 视频在浏览器本地处理，不会上传到服务器。
- 首次使用会从 CDN 加载 FFmpeg WebAssembly，可能需要等待一会儿；PWA 会缓存应用文件。
- 大文件压缩会占用较多内存，建议先用“更小”或 720p 档位。
